13. End-of-waste status bekreftet for brukte materialer  
14. Byggvaredeklarasjoner (BVD) innhentet

20. **Post-completion:** Handover dokumentasjon til klient inkludert compliance records  
21. Garantier og warranty dokumenter arkivert  
22. End-of-life / take-back arrangements dokumentert  
23. **Risk areas & mitigation:** Identifiser høyrisikoområder og hvordan mitigere: **Risk 1: Brannklassifisering failure**Likelihood: Medium-High (brukte tekstiler/møbler ofte ikke tested)  
24. Impact: High (kan stoppe prosjekt, require replacement)  
25. Mitigation:  
26. Test early (4+ måneder før levering)  
27. Have backup suppliers  
28. Budget for brannbehandling/impregnering  
29. Specify "or equal" i contracts  
30. **Risk 2: Ukjente kjemikalier i brukte materialer**Likelihood: Low-Medium (mest i materialer fra før 1980-tallet)  
31. Impact: High (health/safety, legal liability)  
32. Mitigation:  
33. Source fra kjente, moderne produkter (post-2000)  
34. Request material safety data sheets  
35. Test hvis tvil  
36. Avoid unknown-origin products i høy-contact areas  
37. **Risk 3: Bygglov-forsinkelser**Likelihood: Medium (depends on kommun)  
38. Impact: Medium (timeline delay, cost overruns)  
39. Mitigation:  
40. Early engagement med bygglov-myndighet  
41. Pre-application meetings  
42. Hire local consultant familiar with kommun  
43. Build 4-6 weeks buffer i schedule  
44. **Risk 4: Leverandør ikke kan levere dokumentasjon**Likelihood: Medium-High (many gjenbruk operators ikke mature on this)  
45. Impact: High (cannot use materials)  
46. Mitigation:  
47. Specify documentation requirements i RFP/contracts  
48. Vet suppliers before engagement  
49. Have testing budget as backup  
50. Maintain relationships with testing labs  
51. **Risk 5: Political/regulatory shift**Likelihood: Low-Medium (politikk kan endre)  
52. Impact: Variable (could ease or tighten requirements)  
53. Mitigation:  
54. Stay updated via Sweden Green Building Council, Boverket newsletters  
55. Build flexibility into long-term strategies  
56. Maintain both circular and traditional supply chains  
57. **When to engage specialists:Brannkonsulent:** Always for projects \>3000 kvm or high-risk  
58. **Kemikaliekonsulent:** If sourcing pre-1990 materials or unknown origin  
59. **Bygglovskonsulent:** If working in new kommun or complex PBL situation  
60. **LCA/klimatdeklaration specialist:** If required by BBR or voluntary certification  
61. **Kulturmiljö-expert:** If building is listed or i särskilt värdefullt område
