## PROMPT 4: BVB (Byggvarubedömningen)

### Ratingsystem
| Rating | Beskrivelse | Kriterium |
|--------|-------------|-----------|
| **Rekommenderad (A)** | Best-in-class | ≥50% resirkulert, ≥70% gjenbrukbart |
| **Accepterad (B)** | OK å bruke | Møter grunnkrav |
| **Undvik (C)** | Unngå | Inneholder farlige stoffer |

### Sirkulær Økonomi Kriterier
- **Resirkulert innhold:** ≥50% for Rekommenderad
- **Gjenbrukbarhet:** ≥70% av produktet må være gjenbrukbart
- **Resirkulerbarhet:** ≥70% må kunne materialgjenvinnes

### Hospitality-spesifikke Gap
- **Møbler:** Begrenset dekning (fokus på byggematerialer)
- **Tekstiler:** Utenfor BVB scope (bruk Nordic Swan/OEKO-Tex)
- **Belysning:** Varierende dekning
- **Gjenbrukte produkter:** Ingen prosedyre for rating

### Spec-maler (fra dokumentet)
```
Seating: BVB A/B OR equivalent + min 30% recycled + take-back program + Euroclass B-s2,d0
Flooring (textile): BVB A OR Nordic Swan/EU Ecolabel + min 50% recycled + Bfl-s1
Lighting: BVB B+ OR EPD + RoHS + design for disassembly
```

| **LEED** | Bygg | Internasjonal | ⭐⭐⭐ |
| **Miljöbyggnad** | Bygg | Svensk | ⭐⭐⭐⭐ |
| **ISO 14001** | Ledelsessystem | Prosess | ⭐⭐⭐⭐ |
| **EU Ecolabel** | Operasjonell | EU-standard | ⭐⭐ |

### Nordic Swan - Nøkkelkrav
- Må kjøpe øko-merkede møbler og tekstiler
- Kun miljøvennlige rengjøringsprodukter (75%+ øko-merket)
- Ingen PVC i gulv/vegger
- LED-belysning

### EU Ecolabel - Sirkulær Innretning
- **Criterion 52:** 40%+ øko-merkede varige varer
- **Criterion 57:** Donasjon/gjenbruk av møbler og tekstiler (2 poeng)

### Scandic Hotels Benchmark
- 88% Nordic Swan certified
- Siden 1996: -32% vann, -31% energi, -65% CO2e
